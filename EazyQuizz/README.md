# EazyQuizz

## Descriere
Aplicație Console în C# pentru evaluarea cunoștințelor dintr-un domeniu specific.

## Caracteristici
- Teste de cunoștințe
- Autentificare student (nume + parolă)
- Salvare date în fișier text
- Evaluare și scoring automat
- Afișarea rezultatelor

## Structură Proiect
EazyQuizz/
├── Program.cs  
├── Student.cs  
├── Quiz.cs  
├── Question.cs  
├── Answer.cs  
├── QuizResult.cs  
├── students.txt  
├── scores.txt  
└── README.md  

## Clasele Implementate
- **Student** - gestionarea utilizatorilor
- **Quiz** - logica testului
- **Question** - întrebări
- **Answer** - răspunsuri
- **QuizResult** - scor final

## Funcționalități
- Înregistrare student
- Autentificare
- Pornire quiz
- Calcul scor
- Salvare scor în fișier
- Afișare scoruri

## Tehnologii
- C#
- Console Application
- Fișiere text (.txt)
