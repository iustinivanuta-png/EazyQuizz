# EazyQuizz

Aplicatie WPF dezvoltata in C# pentru administrarea si rezolvarea quiz-urilor.

## Functionalitati
- Autentificare student
- Administrare intrebari
- Quiz pe domenii
- Salvare scoruri
- Salvare intrebari in fisier
- Interfata grafica WPF

## Tehnologii utilizate
- C#
- WPF
- LINQ
- File Handling
- OOP
- Collections
